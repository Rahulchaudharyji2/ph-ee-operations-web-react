# Payment Hub EE - NextGen Operations UI (React MVP) 🚀

This repository contains the Next-Generation Operations UI for **Payment Hub EE**, developed as part of the **C4GT DMP 2026** program. This project aims to modernize the existing Angular-based Operations application into a highly modular, performant, and premium React-based ecosystem.

![Version](https://img.shields.io/badge/version-2.1.0--MVP-blue?style=for-the-badge)
![Tech Stack](https://img.shields.io/badge/built%20with-React%2019%20%2B%20Vite%20%2B%20TS-61DAFB?style=for-the-badge)

## 🌟 Overview
The NextGen Operations UI is designed to be the central command center for GovStack and non-GovStack orchestration flows. It provides a state-of-the-art experience for managing G2P, P2G, and Voucher distributions with real-time analytics and role-based access.

### ✨ Key Features
- **Modern Analytics Dashboard:** Real-time visualization of transaction volumes and failure rates using Recharts.
- **Batch Management MVP:** Functional parity with the legacy app, allowing management of bulk orchestration batches.
- **Premium Glassmorphism UI:** A sleek, dark-mode administrative interface designed for the Enterprise Edition.
- **Internationalization (i18n):** Full support for English and French out of the box.
- **Scalable Architecture:** Built using **Feature-Sliced Design (FSD)** patterns to ensure long-term maintainability.

## 🛠️ Tech Stack
- **Core:** React 19, TypeScript, Vite
- **Styling:** Tailwind CSS, Framer Motion (Animations)
- **UI Components:** Shadcn UI, Lucide Icons
- **Data Viz:** Recharts
- **i18n:** react-i18next

## 🚦 Getting Started

### Prerequisites
- Node.js (v18+)
- npm or yarn

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/Rahulchaudharyji2/ph-ee-operations-web-react.git
   cd ph-ee-operations-web-react
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm run dev
   ```
